# weather-app
A professional, real-time weather application built with Streamlit and OpenWeatherMap API. Featuring interactive maps, detailed forecasts, air quality monitoring, and a beautiful user interface.

https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=1200&q=80

✨ Features
🌡️ Current Weather
Real-time temperature, humidity, and pressure

Wind speed and direction

Sunrise and sunset times

Weather conditions with icons

📅 Forecasts
7-Hour Forecast: Hourly weather predictions with icons

7-Day Forecast: Daily weather overview

Interactive Charts: Temperature trends visualization

🗺️ Interactive Maps
Location mapping with markers

Multiple map tile layers

Zoom and pan functionality

Google Maps integration

🌬️ Air Quality Monitoring
Real-time AQI (Air Quality Index)

Health recommendations

Color-coded air quality levels

Pollution level indicators

⚙️ Advanced Features
Search any city worldwide

Favorite locations

Temperature unit conversion (°C/°F)

Dark/Light theme options

Data export (JSON format)

Responsive design
